# 关于MyTomato
- 这是一款大小只有3.3MB的番茄时钟，功能却不少。倒计时、统计数据、徽章系统等等。
- 制作它的初衷是因为我需要一款不会在状态栏吸引我注意力的番茄时钟。
- 因此状态栏图标采用了十分常见的红绿灯形式来进行计时。
- 当然它不止有状态栏，还有一个窗口面板，倒计时、统计数据、徽章都可以在这里看到。
- 其实之前我是有些许将它收费的想法，赚钱嘛，哈哈。但是决定上传到github到时候，我突然意识到自己除了学校里学过的那些课本上的知识，现在所有的知识都是在互联网上学习到的。
- 当然不包括编程，因为我始终没学会编程，没想到吧，不会编程居然还做了个软件。感觉现在真是处在第三次工业节点上，连我这种不会编程的人都能按照自己的想法作出自己的软件了，像不像不会织布的人操控织布机，AI让很多事情方便了起来。
- 我很感激互联网的开源精神，如果没接触到互联网或许我现在在某个山里放牛吧。
- 因此这个软件当然也开源。
- 只是GitHub我也是第一次使用，我觉的有了AI就不应该在说这句话，但是看到AI给出的git繁琐步骤我还是慢慢来吧。有些时候真的是看不进去一点这种信息，或许这是犟人的一点坚持。
- 这个软件肯定会有我没发现的bug，虽然目前为止已经做了几十版了，欢迎大家提交问题，有新的想法也可以提哦。
